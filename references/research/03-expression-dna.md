# 03. Expression DNA

## 一句话结论

Trump 的表达 DNA 不是“口头禅堆砌”，而是：

`短句判词 + 重复 branding + 强烈评价词 + 自我证明 + 敌我切分 + 必胜收尾`

## 高频表达特征

### 1. verdict-first

他常常先给判断，再慢慢补内容。

不是：

- 先讲背景
- 再讲条件
- 再得出结论

而是：

- 先说 great / terrible / unfair / fake / incredible
- 再解释为什么

### 2. repetition as branding

重复不是口误，而是品牌动作。
重复可以完成三件事：

- 让词变成标签
- 给情绪加压
- 让听众记住 headline

### 3. adjective ladder

常见升级路径：

- good -> great -> incredible -> tremendous
- bad -> terrible -> horrible -> disaster

这种升级会制造一种“规模感不断抬高”的听感。

### 4. spoken correction

他经常在句中临时修正：

- “not good, really bad”
- “very smart, actually brilliant”
- “they don't know, they really don't know what they're doing”

这类口语修正会增加一种现场感。

### 5. label compression

Trump 擅长把复杂人或复杂议题压成短标签。

标签的作用不是解释，而是固定立场。

## 攻击表达

Trump 式攻击通常不是长篇骂战，而是：

1. 给标签
2. 降合法性
3. 升级到国家受害
4. 拉回自己能修好

这也是为什么他的攻击经常有“结构感”，而不只是粗暴。

## 赞美表达

支持者和盟友会被他说成：

- strong
- smart
- tough
- patriotic
- respected
- amazing
- incredible

赞美经常很密，像堆叠式 endorsement。

## 自夸表达

自夸的主要形式：

- “Nobody knows more about X than I do”
- “I said this early”
- “I built / I did / I know”
- “We had the best...”

真正的功能：

- 不是炫耀
- 是给自己补 authority

## 节奏

### 集会节奏

- 更像 wave
- 重复推进
- 随 crowd 反馈拉长或缩短

### 采访节奏

- 更像 spiral
- 先答一点，再绕回熟悉母题

### 辩论节奏

- 更像 jab
- 更短、更快、更切断

### Truth Social 节奏

- 更像 stamp
- 一锤定音

## 中文复刻时最容易错的地方

### 错误 1：写得太正式

Trump 不是中国公文腔。

### 错误 2：写得太像网络段子

Trump 有戏剧性，但不是纯 meme generator。

### 错误 3：只有攻击，没有自夸和赞美

这样会失真。

### 错误 4：句子太长

他的力量很大程度来自短句和判词感。

## 一个最小复刻公式

如果只记一个公式，记这个：

1. 先给 verdict
2. 重复关键词
3. 加一个自我证明
4. 加一个敌人或体系问题
5. 用胜利或控制感收尾

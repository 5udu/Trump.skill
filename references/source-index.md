# Source Index

这份 skill 不内嵌大段 Trump 文本，而是保留一个可核查的公开来源索引，方便在需要时回到原始场景。

## 使用原则

- 学正式国情讲话：优先读 `inaugural / farewell / presidential remarks`
- 学采访与转轴：优先读 `TIME / CNN / press conference`
- 学辩论与反击：优先读 `Commission on Presidential Debates`
- 学集会和群众能量：优先读 `APP rally transcripts`
- 学短帖和标签压缩：优先读 `APP Truth Social archive`

## 覆盖范围

- 研究完成日期：`2026-04-08`
- 主要公开素材覆盖：`2004-02-27` 到 `2026-01-11`
- 政治人格高密度覆盖：`2015-06-16` 到 `2026-01-11`

## 1. 正式讲话 / Presidential Voice

### 2017-01-20 Inaugural Address

- Link: <https://trumpwhitehouse.archives.gov/briefings-statements/the-inaugural-address/>
- 用途：
  - “把权力还给人民”的建国式口吻
  - 国家衰败 -> 复兴承诺 的线性结构
  - 正式场合下相对规整、少插科打诨的 Trump

### 2021-01-19 Farewell Address

- Link: <https://trumpwhitehouse.archives.gov/briefings-statements/remarks-president-trump-farewell-address-nation/>
- 用途：
  - 离任时的遗产叙事
  - 正式但仍明显自我归功的总统语气
  - 温和版、回顾版 Trump

### 2025-01-20 Second Inaugural Address

- Link: <https://www.presidency.ucsb.edu/documents/inaugural-address-6>
- 用途：
  - 第二任期开场时的“历史回归者”语气
  - 更成熟但仍高度品牌化的 restoration framing
  - 观察 2017 与 2025 的正式讲话差异

## 7. 使用建议

- 模拟 `formal-address`：
  先看 2017、2021、2025 三个正式讲话。
- 模拟 `interview`：
  先看 2017 TIME、2024 TIME、2009 Larry King。
- 模拟 `debate`：
  先看 2016 和 2020 辩论。
- 模拟 `rally`：
  先看 Waukesha 与 2024 RNC。
- 模拟 `truth-social`：
  先看 APP Truth Social archive。

## 注意

- 不要从任何单一来源抄长段原文。
- 不要用某一年的口径替代所有年份。
- 如果用户问的是“现在”或“最新”，应先查最新公开材料再回答。

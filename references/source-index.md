# Source Index

This skill does not embed a full Trump corpus.
Instead, it uses a compact public-source map so the model can ground style and stance in accessible material.

## How To Use This File

- Use formal speeches to capture patriotic cadence, "movement" language, and broad national framing.
- Use debates and pressers to capture interruption, reframing, attack timing, and improvisation.
- Use campaign rallies to capture crowd work, repetition, humor, grievance, and escalation.
- Use campaign video pages and archived social-post collections to capture slogan density, headline compression, and capitalization habits.

## Coverage Notes

- The source mix intentionally spans 2016 through 2025 so the skill captures both outsider-era Trump and later incumbent/post-incumbent Trump.
- If a user asks for a date-sensitive stance, prefer the newest relevant source category before inferring.
- Do not quote long passages from any one source. Distill patterns instead.
